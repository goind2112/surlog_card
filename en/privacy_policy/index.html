SurLog Privacy Policy

This application collects certain personal data from its users.

We do not use names, Apple ID, or any other personal user data for targeted advertising. Data collected by the app is used solely for app functionality, analytics, subscriptions, and user support.

Summary of the Privacy Policy

Personal data is collected for the following purposes and processed using the following services:

Analytics

  • Personal Data: app usage data, crash and performance data, other diagnostic data, user and device IDs
  • Services used: Amplitude, Sentry

Support

  • Personal Data: data generated by the user during customer support interactions

Notifications

  • Personal Data: user ID
  • Services used: OneSignal

Subscription Management

  • Personal Data: purchase data, user ID
  • Services used: RevenueCat

Additional Information on Personal Data

Push Notifications

The app may send push notifications to the user for alerts and updates.

Types of Collected Data

Personal data collected by the app directly or via third-party services may include:

  • Name
  • Email address
  • Data generated by the user during customer support interactions
  • User and device IDs
  • Purchase data
  • App usage data
  • Crash, performance, and other diagnostic data
  • Permission to send notifications

Methods and Location of Data Processing

The owner implements security measures to prevent unauthorized access, disclosure, modification, or destruction of data. Processing is carried out using computers and IT tools strictly within the purposes of data collection.

Legal Basis for Processing

  • The user has given consent for one or more specific purposes
  • Provision of data is necessary to perform a contract with the user
  • Processing is necessary to comply with legal obligations
  • Processing is related to the legitimate interests of the owner

Data Processing Location

Data is processed at the owner’s offices and other locations where parties involved in processing are located. Data may be transferred to other countries.

Data Retention

Personal data is retained for the period necessary to fulfill the purposes for which it was collected.

Purposes of Processing

  • Ensuring the app functions correctly
  • Compliance with legal obligations
  • Protection of the rights and interests of the owner or users
  • Detection of fraudulent or malicious activity
  • Analytics, device data access, infrastructure monitoring

Infrastructure Monitoring

This service allows monitoring of app components to improve performance and resolve issues.

User Rights

Users have the right to:

  • Withdraw consent for data processing
  • Object to data processing
  • Verify and correct their data
  • Request deletion of their data

Additional Information

  • Personal data may be used in legal proceedings if necessary
  • System logs and IP addresses may be collected for app maintenance
  • The app does not support “Do Not Track”; check the privacy policies of third-party services
  • The policy may be updated; users will be notified on this page
