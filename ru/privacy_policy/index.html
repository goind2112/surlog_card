Политика конфиденциальности SurLog

Это приложение собирает некоторые персональные данные своих пользователей.

Мы не используем имена, Apple ID или любые другие личные данные пользователей для таргетированной рекламы. Данные, собранные приложением, применяются исключительно для работы приложения, аналитики, подписок и поддержки пользователей.

Краткое описание политики конфиденциальности

Персональные данные собираются для следующих целей и обрабатываются следующими сервисами:

Аналитика

  • Персональные данные: данные об использовании приложения, данные о сбоях, производительности и другие диагностические данные, ID пользователя и устройства
  • Используемые сервисы: Amplitude, Sentry

Поддержка

  • Персональные данные: данные, генерируемые пользователем во время обращений в поддержку

Уведомления

  • Персональные данные: ID пользователя
  • Используемые сервисы: OneSignal

Управление подписками

  • Персональные данные: данные о покупках, ID пользователя
  • Используемые сервисы: RevenueCat

Дополнительная информация о персональных данных

Приложение может отправлять пользователю push-уведомления для оповещений и обновлений.

Типы собираемых данных

  • Имя
  • Email
  • Данные, генерируемые пользователем во время обращений в поддержку
  • ID пользователя и устройства
  • Данные о покупках
  • Данные об использовании приложения
  • Данные о сбоях, производительности и другие диагностические данные
  • Разрешение на отправку уведомлений

Способы и место обработки данных

Владелец принимает меры безопасности, чтобы предотвратить несанкционированный доступ, раскрытие, изменение или уничтожение данных. Обработка выполняется с помощью компьютеров и IT-инструментов строго в рамках целей обработки. Доступ к данным могут иметь также сотрудники и подрядчики (администрация, маркетинг, IT-компании и др.) как обработчики данных.

Правовое основание обработки

  • Пользователь дал согласие на обработку данных для одной или нескольких целей
  • Предоставление данных необходимо для исполнения договора с пользователем
  • Обработка необходима для соблюдения юридических обязательств владельца
  • Обработка связана с законными интересами владельца

Место обработки

Данные обрабатываются в офисах владельца и в других местах, где находятся стороны, участвующие в обработке. Возможна передача данных в другие страны.

Срок хранения

Персональные данные хранятся в течение времени, необходимого для достижения целей их сбора.

Цели обработки

  • Обеспечение работы приложения
  • Соблюдение юридических обязательств
  • Защита прав и интересов владельца или пользователей
  • Выявление мошеннической или вредоносной активности
  • Аналитика, доступ к данным устройства, мониторинг инфраструктуры

Мониторинг инфраструктуры

Сервис позволяет отслеживать работу компонентов приложения для улучшения его производительности и устранения сбоев.

Права пользователей

  • Отозвать согласие на обработку данных
  • Возражать против обработки данных
  • Проверять и исправлять свои данные
  • Требовать удаления данных

Дополнительная информация

  • Персональные данные могут использоваться в судебных процессах по необходимости
  • Логи системы и IP-адреса могут собираться для обслуживания приложения
  • Приложение не поддерживает “Do Not Track”; проверьте политику сторонних сервисов
  • Политика может обновляться, пользователи будут уведомлены на этой странице
